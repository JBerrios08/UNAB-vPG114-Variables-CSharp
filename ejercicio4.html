Ejercicio 4 - Promedio de Tres Números

1. Lenguaje Natural

  • Definir tres variables de tipo double para representar notas.
  • Asignar valores a las tres notas.
  • Calcular el promedio de las tres notas.
  • Mostrar el resultado en pantalla.

2. Diagrama de Flujo

3. Pseudocódigo

Algoritmo CalcularPromedio
    Definir nota1, nota2, nota3 Como Reales
    nota1 <- 10
    nota2 <- 8
    nota3 <- 9

    promedio <- (nota1 + nota2 + nota3) / 3

    Escribir "El promedio de las notas es: ", promedio
FinAlgoritmo

4. Código en C#

internal class Program
{
    private static void Main(string[] args)
    {
        /*
         * Promedio de tres números
         * Declara tres variables de tipo double, por ejemplo, nota1, nota2, nota3.
         * Asigna valores a las tres notas.
         * Calcula el promedio de las tres notas y muestra el resultado.
         */

        double nota1 = 10;
        double nota2 = 8;
        double nota3 = 9;

        double promedio = (nota1 + nota2 + nota3) / 3;

        Console.WriteLine("El promedio de las notas ingresadas es " + promedio);
    }
}
