Ejercicio 2 - Área del Círculo

1. Lenguaje Natural

  • Solicitar al usuario el radio de un círculo.
  • Calcular el área del círculo usando la fórmula A = π * radio^2.
  • Mostrar el resultado del área al usuario.

2. Diagrama de Flujo

3. Pseudocódigo

Algoritmo CalcularAreaCirculo
    Escribir "Ingrese el radio del círculo:" 
    Leer radio
    area <- 3.1416 * radio^2
    Escribir "El área del círculo es: ", area
FinAlgoritmo

4. Código en C#

internal class Program
{
    private static void Main(string[] args)
    {
        /*
         * Área del círculo
         * Declara una variable de tipo double para almacenar el radio de un círculo.
         * Usa la fórmula del área del círculo, A = π * radio^2, para calcular el área.
         * Muestra el resultado en consola.
         */

        double radio = 4; // Declarar variable para almacenar el radio del círculo
        double areaCirculo = Math.PI * Math.Pow(radio, 2); // Aplicar la fórmula para calcular el área

        // Mostrar resultado en consola
        Console.WriteLine("El área del círculo es " + areaCirculo);
    }
}
