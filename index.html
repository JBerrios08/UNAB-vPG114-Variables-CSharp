Ejercicio 1

1. Lenguaje Natural

  • Solicitar al usuario que ingrese cinco números.
  • Sumar los cinco números.
  • Mostrar el resultado de la suma al usuario.

2. Diagrama de Flujo

3. Pseudocódigo

Algoritmo SumaCincoNumeros
    Escribir "Ingrese el primer número:"
    Leer numero1
    Escribir "Ingrese el segundo número:"
    Leer numero2
    Escribir "Ingrese el tercer número:"
    Leer numero3
    Escribir "Ingrese el cuarto número:"
    Leer numero4

FinAlgoritmo

4. Código en C#

internal class Program
{
    private static void Main(string[] args)
    {
        /*
         * Suma de 5 números
         * Declarar cinco variables de tipo entero (int).
         * Asigna un valor numérico a cada variable ingresado por el usuario.
         * Muestra en la consola la suma de todas las variables.
         */

        // Declaración de las variables
        int numero1 = 10;
        int numero2 = 20;
        int numero3 = 30;
        int numero4 = 40;
        int numero5 = 50;

        // Cálculo de la suma
        int sumatoria = numero1 + numero2 + numero3 + numero4 + numero5;

        // Mostrar el resultado en consola
        Console.WriteLine("El resultado de la sumatoria de los 5 números es " + sumatoria);
    }
}
